# Resev2 - Horror Adventure

## [>> СКАЧАТЬ ИГРАБЕЛЬНУЮ ДЕМО-ВЕРСИЮ (v0.1.0) <<](https://github.com/Girjey/Resev2/releases/latest)

Resev2 (в разработке)
Психологический хоррор-детектив с point-and-click геймплеем в стиле игр эпохи PS1.

О проекте
Это мой учебный проект, в котором я пробую себя в роли геймдизайнера и разработчика. События игры происходят в отдаленном полицейском участке, где неудачный эксперимент превратил всех сотрудников в жутких созданий. Игрок — единственный выживший оперативник, которому предстоит выяснить причины катастрофы.

Что реализовано в текущей демо-версии (v0.1.0):
Перемещение по коридорам и комнатам полицейского участка.
Базовое взаимодействие с предметами.
Логика запертых дверей.

Управление:
Перемещение и взаимодействие: Мышь
Шрифт: В архиве с игрой лежит шрифт, который нужно установить для корректного отображения текста.

Технологии:
